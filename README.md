EasyRNG
=======

The Portable Ruby, Node.js and Git Package for Front-end Development on Windows.
Windows 下免安裝的 Ruby、Node.js 和 Git 的網頁前端開發包

Usage: 
-------

1. Download it.
2. Unzip it.
3. Double click to execute path.bat.
4. Enjoy.

使用法：
-------

1. 下載 zip 檔。
2. 解壓縮 zip 檔。
3. 滑鼠左鍵點兩下執行 path.bat。
4. 開始使用。
